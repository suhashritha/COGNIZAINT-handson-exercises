# E-commerce Platform Search Function

## Objective
Implement and compare Linear Search and Binary Search algorithms for an e-commerce platform.

## Features
- Product class with product details
- Linear Search implementation
- Binary Search implementation
- Time Complexity Analysis

## Technologies Used
- Java
- Object-Oriented Programming

## Time Complexity

| Algorithm | Best | Average | Worst |
|------------|------|---------|--------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Conclusion
Binary Search is more efficient for large product datasets because it reduces search time significantly compared to Linear Search.
