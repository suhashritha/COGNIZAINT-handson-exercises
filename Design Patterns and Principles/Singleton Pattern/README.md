# Singleton Pattern Example

## Overview

This project demonstrates the implementation of the Singleton Design Pattern in Java. The Logger class is designed to ensure that only one instance of the class exists throughout the application lifecycle. A test class verifies that multiple references point to the same Logger object.

## Features

* Private constructor
* Static instance variable
* Public static getInstance() method
* Singleton verification using object references
* Logging functionality

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Singleton Design Pattern
