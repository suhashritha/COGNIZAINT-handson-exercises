# ORM Learn

This project is developed using Spring Boot, Spring Data JPA, Hibernate, Gradle, and MySQL.

## Technologies Used
- Java 23
- Spring Boot 4.1.0
- Spring Data JPA
- Hibernate
- MySQL
- Gradle

## Features
- Connects to MySQL database
- Uses Hibernate ORM
- Retrieves Country data from the database
- Uses JpaRepository to access data

## Database
Database Name: ormlearn

Table:
- country

Sample Data:

| Code | Name |
|------|------|
| IN | India |
| US | United States of America |

## How to Run

1. Clone the repository.
2. Configure MySQL credentials in `application.properties`.
3. Run the Spring Boot application.
