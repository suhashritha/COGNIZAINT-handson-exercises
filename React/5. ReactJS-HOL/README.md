
# CohortsTracker

A React-based application to track and display different learning cohorts.
The project demonstrates React components, props, CSS Modules, inline styling, and dynamic rendering.

## Features

* Display cohort details
* Reusable React components
* Dynamic rendering using `map()`
* Search cohorts by name
* CSS Module based styling
* Inline styling based on cohort status

## Technologies Used

* React JS
* JavaScript (ES6)
* HTML
* CSS Modules

## Installation

Clone the repository:

```
git clone <repository-url>
```

Navigate to the project folder:

```
cd CohortsTracker
```

Install dependencies:

```
npm install
```

## Running the Application

Start the React development server:

```
npm start
```

The application will run in the browser at:

```
http://localhost:3000
```
