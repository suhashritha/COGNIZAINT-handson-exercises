# Student Management Portal - React Application

## Objective

This project is created to understand React components, component rendering, and the structure of a React application.

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- Node.js
- npm

## Project Description

Student Management Portal is a beginner React application that demonstrates the use of multiple components.

The application contains three components:

1. Home Component
2. About Component
3. Contact Component

Each component displays a separate message.

## Components

### Home Component

Displays:

```
Welcome to the Home page of Student Management Portal
```

### About Component

Displays:

```
Welcome to the About page of the Student Management Portal
```

### Contact Component

Displays:

```
Welcome to the Contact page of the Student Management Portal
```

## Installation

Clone the repository and navigate to the project folder.

Install dependencies:

```bash
npm install
```

## Run Application

Start the React application:

```bash
npm start
```

Open browser:

```
http://localhost:3000
```

The application displays:
