# My First React Application

## Objective
This project is a beginner React application created as part of a hands-on lab to understand the basics of React.

## Technologies Used
- React
- JavaScript
- HTML
- CSS
- Node.js
- npm

## Project Description
This application displays the following heading:

**Welcome to the first session of React**

It was created using `create-react-app`.

## Project Structure

```
myfirstreact/
│── public/
│── src/
│   └── App.js
│── package.json
│── package-lock.json
│── README.md
│── .gitignore
```

## Installation

1. Clone the repository.
2. Navigate to the project folder.
3. Install dependencies:

```bash
npm install
```

4. Run the application:

```bash
npm start
```

5. Open your browser and visit:

```
http://localhost:3000
```

## Output

```
Welcome to the first session of React
```
